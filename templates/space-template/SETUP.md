# SETUP FILE

Setup files are required to start a project, it's a guide on what's needed for a space/project/department/area to function and to grow.

## MINIMUM REQUIREMENTS
- 3-5 Coordinators / Managers / Directors. Active Leaders.
- 3 Active members.
- Monthly updated PROJECT file status.
- Published open monthly online meeting.
- Monthly updated Project README file.
- Updated project Mission and objectives.
- Rotatory leadership roles.
